# 🌤️ GlowMinder

> Weather-adaptive skincare reminder system — built for Coding Camp 2026 
> powered by DBS Foundation (Team CC26-PSU022)

---

## 🧴 About

GlowMinder helps users maintain consistent skincare routines by delivering 
smart reminders based on real-time weather conditions. Instead of recommending 
new products, GlowMinder maximizes what users already own — adapting 
suggestions to temperature, humidity, and UV index.

---

## ✨ Features

- 📦 Manage your personal skincare product collection
- 🕐 Categorize products by usage time (morning, night, special treatment)
- 🌡️ Real-time weather integration (OpenWeatherMap / BMKG API)
- 🤖 AI-powered automatic product classification (TensorFlow NLP)
- 🔔 Smart weather-based reminders with natural language notifications
- 📊 Weather-skincare correlation dashboard (Streamlit)
